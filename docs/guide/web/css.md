# css

## 选择器

-   标签选择器

```css
div {
	width: 300px;
	height: 300px;
	background-color: red;
}
p {
	text-indent: 2em;
	color: blue;
}
span {
	letter-spacing: 5px;
	font-size: 20px;
}
/* 通用选择器 */
* {
	magrin: 0;
	padding: 0;
}
```

-   类选择器

```css
.box {
	width: 300px;
	height: 300px;
	background-color: red;
}
p.des {
	text-indent: 2em;
	color: blue;
}
```

-   id 选择器

```js
<div class="container">
    <article>
        <h1>Napoléon Bonaparte</h1>
        <p>Adversity is the midwife of genius.</p>
    </article>
</div>

#box {
	width: 300px;
	height: 300px;
	background-color: red;
}
#des {
	text-indent: 2em;
	color: blue;
}
```
